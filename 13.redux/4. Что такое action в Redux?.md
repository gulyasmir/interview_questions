**Ответ:**
Action — это обычный JavaScript-объект, который содержит:

* **type** (обязательно) — строка, описывающая действие.
* **payload** (опционально) — данные, необходимые для выполнения действия.

```
const increment = {
  type: 'INCREMENT'
};

const addUser = {
  type: 'ADD_USER',
  payload: { id: 1, name: 'John Doe' }
};

```
